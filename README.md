# android-camera

(Work in progress)

Android CameraX demo with OpenCV4 and TesorFlow Lite

```
[Image sensor]-->[CameraX]-->[OpenCV4]--+-----------------------+
                     |                  |                       |
                     +------------------+--->[TensorFlow Lite]--+---> Final output
```

## Filters with OpenCV4

- Color filter
- Optical flow
- Difference extraction
- Face recognition
- Intrusion detection

## TensorFlow Lite

- Object detection
